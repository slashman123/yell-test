### Задание 3
В принципе отработает вот такой запрос:

```sql
SELECT
	DISTINCT m.type,
	(SELECT value FROM `data` d WHERE d.type = m.type ORDER BY `date` DESC LIMIT 1) AS value
FROM
  `data` m
```
